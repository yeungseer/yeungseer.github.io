nginx and host

nginx和host的配合，当初折腾了很久的时间，后来发现启动nginx要用sudo，host可以在某个文件夹下的hosts文件上修改，以下为具体过程以及相关命令
nginx中nginx.conf文件中的具体修改就不做赘述了=> /usr/local/etc

  • 启动 sudo nginx
  • 重新加载 sudo nginx -s reload
  • 退出 sudo nginx -s quit
